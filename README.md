> **Everything you need to build and ship AI products. From prototype to production in minutes.**

## 🚀 Quick Start with CLI

```bash
# Install CLI
npm install -g @prompt-stack/prompt-stack

# Create new project
prompt-stack create my-saas-app
cd my-saas-app

# Start development
prompt-stack dev
# ✅ Running at http://localhost:3000 (14 seconds)
```

## 🎯 What You Get

**Instant AI-powered SaaS starter** with:
- ✅ **Authentication** - Supabase with role-based access
- ✅ **AI Integration** - OpenAI, Anthropic, DeepSeek, Gemini
- ✅ **Payments** - Stripe & Lemon Squeezy ready
- ✅ **Database** - PostgreSQL with vector search
- ✅ **Frontend** - Next.js 15 with TypeScript
- ✅ **Backend** - FastAPI with async/await
- ✅ **Docker** - One-command development environment

## 🚀 Quick Start Options

### Option 1: CLI Tool (Recommended)
```bash
npm install -g prompt-stack-cli
prompt-stack create my-app
cd my-app && prompt-stack dev
```

### Option 2: Manual Setup
```bash
git clone https://github.com/prompt-stack/prompt-stack.git
cd prompt-stack/studio
./setup.sh && make dev
```

## 📊 Performance

- **Demo Mode**: 14 seconds to running app
- **Production Setup**: 4-5 minutes total
- **CLI vs Manual**: 30 seconds vs 3-5 minutes
